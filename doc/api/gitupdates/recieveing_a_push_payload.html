GitUpdates API

GitUpdate resource

Recieveing a push payload

POST /git_updates

Parameters

Name Description
commits Commits, as a Array, for example: [ { "sha": "f66997d4630d353901a64f39df5f92d22e4bc634", "message": "FEAT: Support Android 8.1 devices \n\nRef: #sp-131", "date": "2018-05-14T16:00:49Z", "author": { "id": 21431007, "name": "ahmed", "email": "ahmed@suitepad.de" } }, { "sha": "fbea31d5a04cd9e238fd45f3f27f314ebf707479", "message": "FIX: hide url not being passed to PDF URL\n\nRef: #sp-124", "date": "2018-05-27T16:00:49Z", "author": { "id": 21431007, "name": "ahmed", "email": "ahmed@suitepad.de" } } ]
repository Repository, as a Hash, for example: { "id": 1354927974, "name": "suite_apk" }
pushed_at Pushed At, as a String, for example: "2018-05-29T20:18:44Z"
pusher Pusher, as a Hash, for example: { "id": 21431007, "name": "ahmed", "email": "ahmed@suitepad.de" }
